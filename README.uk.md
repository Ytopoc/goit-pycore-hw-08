# Адресна книга - постійне зберігання через pickle

CLI-адресна книга, що переживає перезапуски. Розвиток попередньої ітерації з додаванням серіалізації через `pickle` - контакти зберігаються у `addressbook.pkl` при виході та підвантажуються при наступному запуску.

## Стек

- Python 3.12 (тільки стандартна бібліотека)

## Команди

| Команда | Дія |
|---------|-----|
| `add <ім'я> <телефон>` | Створити контакт |
| `change <ім'я> <старий_тел> <новий_тел>` | Оновити телефон |
| `phone <ім'я>` | Показати всі телефони контакту |
| `add-birthday <ім'я> <DD.MM.YYYY>` | Встановити день народження |
| `show-birthday <ім'я>` | Показати день народження контакту |
| `birthdays` | Дні народження на найближчі 7 днів (вихідні пропускаються) |
| `all` | Показати всі контакти |
| `close` / `exit` | Зберегти книгу та вийти |
| `hello` | Привітання |

## Запуск

```bash
python hw8.py
```

Стан зберігається у `addressbook.pkl` поряд зі скриптом.
